
# Bedrock RAG Starter
Open-source starter kit for building enterprise knowledge chat using **Symfony, AWS Bedrock, and MongoDB Atlas Vector Search**.

---

## 🚀 What This Project Does

This project helps you build an AI-powered knowledge assistant that can:

* 📄 Ingest documents (PDF, text, etc.)
* ✂️ Chunk and embed content using AWS Bedrock
* 🔍 Perform semantic search with MongoDB Atlas
* 💬 Answer questions using retrieved knowledge
* ⚡ Stream responses in real time
* 🔗 Show citations from source documents

---

## 🏗️ Architecture

* **Frontend / Backend:** Symfony (PHP)
* **AI Service:** Python (FastAPI)
* **LLM + Embeddings:** AWS Bedrock
* **Vector DB:** MongoDB Atlas
* **Infrastructure:** Docker
